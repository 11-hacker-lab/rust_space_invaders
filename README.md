# Rust Space Invaders 🛸

A lightweight, fast, and polished clone of the classic arcade game **Space Invaders**, built from scratch in Rust using the `macroquad` game engine.

## Features
* **Dynamic Difficulty:** Aliens accelerate as their numbers decrease, making the endgame intense and challenging.
* **Customizable Difficulty:** Select from multiple difficulty levels to match your playstyle.
* **Color Themes:** Full customization of colors for the player, enemies, and bullets.
* **Remappable Controls:** Switch control schemes easily for a more comfortable gaming experience.
* **Continuous Firing:** Hold down the attack key with a built-in weapon cooldown system.
* **Zero-Lag Performance:** Extremely low memory footprint and high FPS thanks to Rust's zero-cost abstractions.

---

## Controls & Settings

| Key | Action / Feature |
| :--- | :--- |
| `Left Arrow` / `Right Arrow` | Move Ship Left / Right (Default) |
| `Left Ctrl` (Hold) | Continuous Shoot (Default) |
| `Enter` | Start Game (Menu) |
| `Esc` | Return to Main Menu (Game Over screen) |
| **In Settings** | Remap Controls, Change Game Difficulty, Customize Object Colors |
