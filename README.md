# Rust Space Invaders 🛸

A lightweight, fast, and polished clone of the classic arcade game **Space Invaders**, built from scratch in Rust using the `macroquad` game engine.

## Features
* **Continuous Firing:** Hold down the attack key with a built-in weapon cooldown system.
* **Smooth Controls:** Responsive movement tailored for modern keyboard layouts.
* **Zero-Lag Performance:** Extremely low memory footprint and high FPS thanks to Rust's zero-cost abstractions.

---

## Controls

| Key | Action |
| :--- | :--- |
| `Left Arrow` / `Right Arrow` | Move Ship Left / Right |
| `Left Ctrl` (Hold) | Continuous Shoot |
| `Enter` | Start Game (Menu) |
| `Esc` | Return to Main Menu (Game Over screen) |

---

### Prerequisites

Make sure you have [Rust and Cargo](https://www.rust-lang.org/tools/install) installed on your system.
